= Royal Hemlock

image:https://melpa.org/packages/royal-hemlock-theme-badge.svg["MELPA", link="https://melpa.org/#/royal-hemlock-theme"]

This package provides the following themes for Emacs 24+:

* **Royal Hemlock** {em} A soothing *light-theme* for Emacs, focused on *royal blue* colour against a warm white background.

* **Mono Hemlock** {em} A soothing *light-theme* for Emacs, focused on *dark grey* colour against a warm white background. A variant of Royal Hemlock.
